
# Minhas Tarefas

Minhas Tarefas é uma plataforma web e mobile para gerenciamento de tarefas, organizada por dias da semana. Permite a adição de novas tarefas em dias específicos e fornece feedback sobre a porcentagem de tarefas concluídas.
